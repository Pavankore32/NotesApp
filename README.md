# NotesApp

A simple **Java File I/O Notes Application** that allows you to:
- Add notes
- View saved notes
- Clear all notes

All notes are stored in a local `notes.txt` file using Java File Handling.

---

##  Features
- **Add Note** ‚Üí Append new text into `notes.txt`
- **View Notes** ‚Üí Display all notes saved in the file
- **Clear Notes** ‚Üí Wipe the file contents
- **Exit** ‚Üí Close the application
